Hexo&Github流程

  • hexo init必须在空目录下执行，会生成一个.gitignore文件
  • git init会创建.git/目录
  • hexo-deployer-git会创建.deploy_git/目录,并且初始化一个master分支，将本地public的内容发布到这里，并发布到github仓库origin master分支。
